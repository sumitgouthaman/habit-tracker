# Habit Tracker PWA

A simple and effective Habit Tracker built with React, Vite, and Firebase. Works offline and installs as a PWA.

> Vibecoded using **Antigravity IDE**.

## Features

-   **Smart Tracking**: Daily, Weekly, and Monthly habits with auto-reset.
-   **Flexible Goals**: Track binary completion (Done/Not Done) or specific counts (e.g., 5 cups of water).
-   **Visual Stats**: View streaks and completion progress.
-   **Offline Ready**: Works without internet connection.
-   **Clean UI**: Dark mode design with glassmorphism elements.

## Tech Stack

-   **Frontend**: React, Vite
-   **Backend**: Firebase (Firestore, Auth, Hosting)
-   **Styling**: CSS Modules
-   **Icons**: Lucide React

## Getting Started

1.  **Clone & Install**:
    ```bash
    git clone <repo-url>
    cd HabitTracker
    npm install
    ```
2.  **Run Locally**:
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173`.
